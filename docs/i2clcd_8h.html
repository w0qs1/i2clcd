Detailed Description

A Library for Interfacing AVR with LCDs that have I2C bus.

#include "i2clcd.h"
i2clcd.h
A Library for Interfacing AVR with LCDs that have I2C bus.

Basic functions for use with HD44780 based LCDs that use PCF8574 I/O expander.

Requires: I2C Master library by Peter Fleury pfleu.nosp@m.ry@g.nosp@m.mx.ch

Author
Sandeep Kumar http://www.github.com/w0qs1
Copyright
(C) 2021 Sandeep Kumar, GNU General Public License Version 3
Usage Example:
#include <avr/io.h>
#include <util/delay.h>
#include "i2cmaster.h"
#include "i2clcd.h"
#define LCD_ADDRESS 0x4E // Address of the PCF8574
#define BACKLIGHT 1 // Backlight ON
int main(void) {
lcd_init(LCD_ADDRESS, BACKLIGHT); // Initialize the LCD with BACKLIGHT ON
lcd_clear(); // Clear the Display
lcd_setcursor(0, 0); // Set the cursor to first line, first column
lcd_print("Hello, World!"); // Print "Hello, World!" on the LCD
}
